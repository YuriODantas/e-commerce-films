# e-Films
This is a film e-commerce project with a simple and intuitive interface.

## Libs and Frameworks
In this project we used these tools:

- ReactJs
- Styled-components
- React-router-dom
- Json-server
- React-query
- Axios
- Eslint
- Vite

## Running project

To run the project, use the following commands:

step 1: Clone this project in you computer.
```
git clone urlThisProject
```

step 2: Install all dependencies.

```
npm i
# or
pnpm i
```

step 3: Run project and Json-server
```
npm run dev
# or
pnpm run dev

# run json-server

npm run dev:server
# or
pnpm run dev:server
```
